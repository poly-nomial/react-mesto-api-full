# Проект Mesto бэкенд

Бэкенд проекта "Место". Прописанны пути и контроллеры, с помощью которых осуществляется создание и обновление профилей пользователей, создание и удаление карточек, постановка и снятие лайков.

## Технологии

Express
MongoDB
Mongoose
